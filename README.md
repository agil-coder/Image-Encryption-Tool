# Image-Encryption-Tool
🖼️ Python Image Encryptor
A lightweight CLI tool that uses bitwise XOR operations to encrypt and decrypt images. This project demonstrates Python scripting, NumPy array manipulation, and basic cryptographic concepts.
🧠 Technical Challenges & Learnings1. Bitwise Manipulation vs. Image QualityChallenge: How to hide data in an image without making it look "glitched" or corrupted to the human eye.Solution: I implemented Least Significant Bit (LSB) substitution. By only modifying the last bit of a pixel's RGBA value (e.g., changing 255 to 254), the visual change is less than $0.4\%$, making the encryption invisible to the naked eye while remaining perfectly readable by the script.2. Data Loss in CompressionChallenge: Initially, saving encrypted images as .jpg caused the secret message to disappear.Solution: I discovered that JPEG uses lossy compression, which tweaks pixel values to save space, destroying the hidden bits. I switched the output format to .PNG, which uses lossless compression, ensuring every bit of the secret message remains intact.3. Computational Efficiency with NumPyChallenge: Using standard Python loops to iterate over millions of pixels in a high-resolution photo (like a 12MB RAW file) was extremely slow.Solution: I utilized NumPy vectorization. By treating the image as a multi-dimensional array, I could apply bitwise XOR operations across the entire dataset simultaneously, reducing processing time from seconds to milliseconds.
